User Stories

  1. As an account holder, I want to ask questions, So that the participants can find and answer questions matching the right answer.

  2. As an acount holder, I want to have just one right answer out of four alternatives, So that in the end of the exam/test it will show the total score.

  3. As an admin, I want to have the possibility to save the tests as an template, So that the test can be used another time.

  4. As a guest, I want to have my own profile with my picture, mail and phone, So that I can use my profile to take several tests in the future.

  5. As a guest, I want to get an email with a certificate in PDF, So that i can keep my final score if i past the exam.
